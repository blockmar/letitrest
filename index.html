Let it REST

Lightweight REST mapper for a Controller and View URL model.

Let it REST is an attempt to create a REST-ful framework with a very light footprint. The main URL handler uses a http servlet to get the requests. Mappings are annotation based and the goal of the project is to support Injection for configuration. A basic template engine is provided - using Freemarker. This is a work in progress, please se the Examples catalog for documentation. More will follow.

Dependencies

  • Log4j
  • Javax-inject
  • Jackson
  • Freemarker
  • Servlet API 2.5
For testing:
  • Junit
  • EasyMock
  • Jetty

Install

The project uses Maven for dependencies.
